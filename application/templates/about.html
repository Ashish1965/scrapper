ML-OOPs Dashboard

ML-Oops is a Open-source and new generation dashboard which is specially designed to those person who want to analyze their Data at beginner level. This Dashboard various Functionality to user such as scrapping of Table Data from perticular site such as record of cricket, hocket data etc. Here we have a choice either we scrap the data from the Website or upload the csv or excel files in this Dashboard. You can also able to access various graphs, Correlation matrices, and also able to predict the future result through this Dashboard in the Service Tab.

  • There is Huge Concepts of Data Science hidden behind the background of this Dashboard.
  • You are not able to Access the services of this Dashboard without Login
  • The Service Sections of this Dashboard the analysis of data which is previously uploaded or accessed.
  • If you are not uploaded or accessed any dataset previously then it shows the analysis of Sampledata which is The Hockey dataset that we have take from online website which provides a dataset for scrappers to practice.
